Scoring Process

  1. 1
    Data Collection: Scrape listing details including price, year, mileage, and condition
  2. 2
    Normalization: Apply winsorization to handle outliers, then normalize each metric to 0-1 scale
  3. 3
    Condition Parsing: Analyze Danish condition text using fuzzy matching to determine quality score
  4. 4
    Weighted Scoring: Combine normalized metrics using configurable weights to produce final 0-100 score
